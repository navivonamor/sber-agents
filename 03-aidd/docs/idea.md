# Идея: LLM-ассистент "Кулинарный помощник"

## Описание

LLM-ассистент должен быть выполнен в виде Telegram-бота.

## Основная задача

Основной задачей бота является вести диалог с пользователем и отвечать на его вопросы с ролью **Кулинарный помощник**, заданной в системном промпте LLM.

## Компоненты

- Telegram-бот для взаимодействия с пользователями
- LLM-модель для генерации ответов
- Системный промпт с ролью "Кулинарный помощник"
